 ### ツール説明
 　
  《VirtualBox（ｳﾞｧｰﾁｬﾙ ﾎﾞｯｸｽ）》
  　　仮想環境を入れておくための箱。
    　ここにLinuxやWindowsなどの仮想OSをセッティングする。
  
  《Vagrant（ﾍﾞｲｸﾞﾗﾝﾄ）》
  　　仮想環境の構築を手伝ってくれるツール。
    　構築に必要だった複雑なコマンドを集約し、簡易に環境構築をしてくれる。
     
   《Visual Studio Code（ｳﾞｨｼﾞｭｱﾙ ｽﾀﾁﾞｵ ｺｰﾄﾞ）》
   　　マイクロソフトにより開発されたオープンソースのコードエディタ。
     　Gitクライアントの統合機能もある。
